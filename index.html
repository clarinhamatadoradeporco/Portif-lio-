Minha Biblioteca: Uma Webpage Personalizada

Este projeto é um jogo do Pac-Man desenvolvido com JavaScript utilizando a biblioteca p5.js. O jogo recria a clássica experiência arcade, com gráficos personalizados, movimentação suave e lógica de colisão. Todo o visual foi desenhado com recursos da p5.js, e o código foi estruturado para facilitar a leitura e futuras melhorias. A estética do jogo foi pensada para ser envolvente e nostálgica.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo. Para o desenvolvimento do jogo PAC-MAN, utilizei JavaScript com a biblioteca p5.js, que permite criar gráficos interativos e animações. Além disso, o CSS foi empregado para estilizar visualmente a página, controlando cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do projeto e o jogo PAC-MAN desenvolvido com JavaScript e a biblioteca p5.js. A área principal do jogo é renderizada em um canvas interativo, onde o jogador controla o personagem PAC-MAN em tempo real. Elementos de controle e informações adicionais são estruturados em contêineres dedicados. O design responsivo é garantido pelo uso da meta tag viewport e de um layout flexível, que se adapta a diferentes tamanhos de tela, oferecendo uma boa experiência em dispositivos variados.

Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma aplicação interativa e divertida chamada projeto Shurek, que apresenta um personagem com aparência inspirada no famoso ogro verde. A proposta é simples, mas envolvente: o personagem acompanha o movimento do cursor do mouse com os olhos, criando uma sensação de interação direta com o usuário.

O ator principal do projeto é o "Shurek", que possui scripts programados para detectar a posição do mouse e ajustar dinamicamente a direção do olhar. Esses scripts são ativados automaticamente ao iniciar o programa (ao clicar na bandeira verde) e permanecem em execução contínua, permitindo que o personagem siga o movimento do cursor em tempo real. Além disso, o visual do personagem é colorido de verde, reforçando a referência divertida ao famoso personagem do cinema.

O Projeto Shurek tem como objetivo explorar conceitos básicos de interatividade e rastreamento de movimento em ambientes de programação visual, oferecendo uma introdução lúdica à lógica de controle e resposta a eventos em tempo real.
